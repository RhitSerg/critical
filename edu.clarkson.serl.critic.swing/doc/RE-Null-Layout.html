Null Layout

I see you are using null layout for a container created in your code [Take Me There]. For any widget to be displayed, it must have a size and location determined somehow. One way to do this is by directly calling JComponenet.setBounds() or JComponent.setSize() and JComponent.setLocation() on the component in a container managed without a layout manager (JPanel.setLayout(null)). Alternatively, the size and location of widgets can be calculated automatically using an algorithm, which is also known as LayoutManger. The consequences of using absolute layout are:

  1. Resizability: The widgets managed by absolute layout will remain in its original size and location when the frame is resized.
  2. Portability: The GUI may not look the same in different platforms.
