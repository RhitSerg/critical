How to Use JFrame

You are using a JFrame in your code [Take Me There]. However, you have not yet configured it properly. Here are few important steps that you should consider while using a JFrame:

  1. Initialize the frame, possibly with a title.
  2. Get hold of its content pane to add new components.
  3. Decide if the default layout manager (BorderLayout) of the JFrame is suitable for your needs. If not, choose another layout manager and set it to the content pane.
  4. Add components to the content pane.
  5. Use pack() to compute the size of the frame.
  6. Set the default close operation behavior.
  7. Set the frame visible.

The following code snippet shows these steps:

public class Test {
	public static void main(String[] args) throws Exception {
		// Step 1: Create a JFrame with a title
		JFrame frame = new JFrame("Test Application");
		
		// Step 2: Get hold of the content pane, which is a JPanel object
		JPanel panel = (JPanel)frame.getContentPane();
		
		// Step 3 and 4: We will use the default BorderLayout, so lets just add components
		JTextArea textArea = new JTextArea("Hello World!", 5, 15);
		panel.add(textArea, BorderLayout.CENTER); // Add to the Center location
		
		JButton button = new JButton("Click Me!");
		panel.add(button, BorderLayout.PAGE_END); // Add to the SOUTH or PAGE_END location
		
		// Step 5: Lets ask frame to compute its size
		frame.pack();
		
		// Step 6: Lets tell the frame to terminate the application when it closes
		frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
		
		// Step 7: Lets show the window
		frame.setVisible(true);
	}
}
