Tutorial: Setting Text Position in a JButton or JLabel

Key facts:

1) When setting the text position, you must set the values of both the horizontal and vertical positions using the setHorizontalTextPosition() and setVerticalTextPosition() APIs.
2) There are nine different possible positions for the text relative to the icon: over the icon, north, south, east, west, northwest, northeast, southeast, and southwest. The picture below displays the nine different positions:

JButton Example

Text Position code to implement picture of output
Northwest northwestButton.setHorizontalTextPosition(AbstractButton.LEFT);
northwestButton.setVerticalTextPosition(AbstractButton.TOP);

northButton.setVerticalTextPosition(AbstractButton.TOP);

northeastButton.setVerticalTextPosition(AbstractButton.TOP);
