Tutorial: How to use Flow Layout

Key facts:

1) FlowLayout is the default layout manager for JPanel objects.
2) FlowLayout puts components in a row, sized at their preferred size.
3) If the horizontal space in the container is too small to put all the components in one row, FlowLayout will use multiple rows.
4) If the container is wider than necessary for a row, then the components will be aligned according to the specified alignment. The default alignment is centered horizontally.
5) Alignment should be specified when creating the FlowLayout object.
6) The flow direction is determined by the container's componentOrientation property. Default orientation is left to right.
7) Spacing between components can be specified by using the setHgap() and setVgap() APIs. Default spacing is 5 pixels for both Horizontal and Vertical spacing.

Constructors

Constructor Purpose
FlowLayout() Constructs a new FlowLayout object with a centered alignment and horizontal and vertical gaps with the default size of 5 pixels.
FlowLayout(int align) Constructs a new FlowLayout with the alignment specified and horizontal and vertical gaps with the default size of 5 pixels. The five possible alignments are: FlowLayout.LEFT, FlowLayout.LEADING, FlowLayout.CENTER, FlowLayout.TRAILING, and FlowLayout.RIGHT.
FlowLayout(int align, int hgap, int vgap) Constructs a new FlowLayout object with the specified alignment and specified horizontal and vertical gap between components. The hgap and vgap arugments specify the number of pixels to put between components.

Component Orientation

Orientation Code To Achieve Desired Orientation
Left to Right container.setComponentOrientation(ComponentOrientation.LEFT_TO_RIGHT)
Right to Left container.setComponentOrientation(ComponentOrientation.RIGHT_TO_LEFT)

Spacing Between Components

Spacing Code To Achieve Desired Spacing
Set horizontal spacing between components layout.setHgap(integer value)
Set vertical spacing between components layout.setVgap(integer value)

Alignment of Components

Alignment Description Picture of Alignment
FlowLayout.CENTER This value indicates each row of components should be centered
FlowLayout.LEFT This value indicates that each row of components should be left-justified
FlowLayout.RIGHT This value indicates that each row of components should be right-justified
FlowLayout.LEADING This value indicates that each component should be justified to the leading edge of the containers orientation. In this example, the orientation is right to left, so the leading edge is the right side
FlowLayout.TRAILING This value indicates that each component should be justified to the trailing edge of the containers orientation. In this example, the orientation is right to left, so the trailing edge is the left side

Notes:

1) The images for FlowLayout.LEADING and FlowLayout.TRAILING are done width a right to left orientation to show the difference between FlowLayout.LEADING and FlowLayout.LEFT and FlowLayout.TRAILING and FlowLayout.RIGHT, which is not clear when using a left to right orientation.
2) FlowLayout.LEFT and FlowLayout.TRAILING produce the same output with a left to right orientation. FlowLayout.TRAILING and FlowLayout.RIGHT also produce the same output with a left to right orientation.
3) The spacing between the edge of the component and the first component in the row is determined by the hgap and vgap properties.

How to Increase the Margin

If you wanted to increase the margin between the edge of the container and the first component there are 2 main ways to do it:

Way to Increase Margin Code Example
Use box.createHorizontalStrut(value) contentPane.add(Box.createHorizontalStrut(50)); //first thing added
Add a blank JLabel of the desired width JLabel blankLabel = new JLabel();
blankLabel.setPreferredSize(new Dimension(50,25));
contentPane.add(blankLabel); //first thing added
